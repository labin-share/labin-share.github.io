1. Spring Cloud

本文仅仅是概括Spring Cloud所用到的组件，并不涉及具体讨论。

spring的微服务框架，提供了全套的分布式系统解决方案。比如服务的注册发现，分布式session，熔断降级等。

Spring cloud是基于spring boot进行开发的，所以Spring cloud用到的组件都有响应的starter， 很方便能够快速构建一个项目。

一般来说，Spring cloud用到的组件如下所列：

  1. Zuul。用于搭建网关
  2. Hystrix， hystrix stream, turbine stream， hystrix dashboard。提供熔断，限流，降级等机制。
  3. Feign。一个http客户端
  4. Ribbon。提供负载均衡的功能。
  5. Eureka。服务注册和发现中心。
  6. Spring cloud config 和 spring cloud bus。配置中心，提供配置文件的集中管理，动态切换等功能。
  7. sleuth 和 zipkin。服务调用链路的追踪。
